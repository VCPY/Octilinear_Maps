Class OctiNode

A single node in the graph. This shares a location with 8 other OctiNodes on the same GridNode. It is connected to all OctiNodes on this GridNode.

It is either a sink node, which are used as endpoints for routing, or a port node, which means that it has an additional edge to the port node of the neighbouring GridNode.
